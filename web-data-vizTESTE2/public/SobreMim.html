Caso contrário, redireciona o usuário para a página de login (../login.html). limparSessao(): Esta função limpa todos os dados armazenados no sessionStorage e, em seguida, redireciona o usuário para a página de login (../login.html). aguardar(): Esta função é usada para mostrar uma mensagem de carregamento na interface do usuário enquanto algum processo está ocorrendo em segundo plano. Ela torna visível uma div com o id div_aguardar, que geralmente contém uma animação de carregamento. finalizarAguardar(texto): Esta função é chamada para esconder a mensagem de carregamento após a conclusão do processo em segundo plano. Ela esconde a div com o id div_aguardar e, opcionalmente, exibe uma mensagem de texto no elemento com o id div_erros_login. Basicamente, estas funções são utilizadas em um sistema web para gerenciar sessões de usuário, fornecer feedback visual durante operações assíncronas e tratar erros de login. Por exemplo, validarSessao() verifica se o usuário está logado, limparSessao() faz logout do usuário, aguardar() mostra um indicador de carregamento durante operações demoradas e finalizarAguardar() esconde o indicador de carregamento após a conclusão da operação.
